Description of the Dataset

The dataset includes medical characteristics like:
Age
Sex
type of chest pain
Blood pressure at rest
level of cholesterol
Blood sugar during fasting

Reaching the maximum heart rate

Angina brought on by exercise

ST depression

The quantity of large vessels

Thalassemia

Target (0 = No illness, 1 = disease)

Whether the patient has heart disease is indicated by the target variable.

**Used Libraries & Technologies
**
Python

Pandas for manipulating data

Numerical operations, or NumPy

Matplotlib (visualization of data)

Seaborn (High-level visualization)

Machine learning models using Scikit-Learn

Kaggle (environment for development)
